# My Quick Links Site


This is a simple, clean, and responsive website containing quick access buttons to various links, including GoFile, Discord, YouTube, and Roblox games.


## Features


- **Responsive design**: Works on desktop and mobile.
- **Quick access buttons**: Open links in a new tab.
- **Custom styles**: Modern, glassmorphism-inspired card layout.

## How to Use


1. Clone this repository or download the files.
2. Place the `index.html` at the root of your repository.
3. Enable **GitHub Pages** in repository settings (select the branch and root folder).
4. Open the GitHub Pages link to see your site live.
